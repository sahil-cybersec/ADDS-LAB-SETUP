Active Directory Domain Services (ADDS) Lab
Project Overview
This repository demonstrates my hands-on experience in deploying and managing an Active Directory Domain Services (AD DS) environment in a virtual lab setup.
The purpose of this lab was to understand the installation, configuration, and administration of Active Directory in a Windows Server environment.

Tasks Performed
•	Installed Windows Server 2016.
•	Configured a Static IP Address.
•	Installed the Active Directory Domain Services (AD DS) role.
•	Promoted the server to a Domain Controller.
•	Configured DNS during domain promotion.
•	Created Organizational Units (OUs).
•	Created and managed domain users and groups.
•	Joined a Windows client machine to the domain.
•	Verified communication between domain controller and client systems.
