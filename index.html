Next-Gen Active Directory Infrastructure Deployment

Executive Summary

Welcome to the digital nerve center! This environment establishes a robust, four-node Active Directory architecture for the work.com domain. Built entirely on Windows Server 2016 Standard Evaluation (1607), the topology ensures high availability through an Additional Domain Controller (ADC), secure branch-office operations via a Read-Only Domain Controller (RODC), and domain compartmentalization using a Child Domain Controller (CDC).
